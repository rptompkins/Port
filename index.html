Loopr

Loopr is a music pad HTML5 Web Audio app. Mapped to each key are specific sounds, loops, and samples, all synchronized and beat-matched under the selected tempo. The user selects which sounds to combine, forming their own beat and changing it or layering it as they choose. Inspired by the Novation Launchpad hardware controller.

Built with: HTML5, CSS3, and Javascript

The Wall

The Wall was a LAMP stack project built to function as a message board. It allowed users to create an account and store their information in a database. They could then log on and view their profile, as well as go to other profile pages and leave messages and comments.

Built with: HTML5, CSS3, PHP/CodeIgniter, and SQL
